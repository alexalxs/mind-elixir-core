🧠 Teste de Integração - Assistente IA Mind Elixir

📋 Instruções de Teste

  1. Abrir o projeto: Acesse http://localhost:23333/
  2. Acessar o Assistente IA:
    • Clique com o botão direito em qualquer nó do mapa mental
    • Selecione "🤖 Assistente IA" no menu de contexto
  3. Testar os modos:
    • 🌱 Expandir: Gera subtópicos para o nó selecionado
    • 💡 Sugerir: Sugere ideias relacionadas
    • 📝 Resumir: Cria um resumo do ramo
    • ❓ Perguntas: Gera perguntas exploratórias
    • ✨ Personalizado: Permite usar um prompt customizado
  4. Aplicar sugestões: Clique no botão + ao lado de cada sugestão para adicioná-la como filho do nó

🔍 O que verificar:

  • ✅ O painel do assistente IA aparece corretamente
  • ✅ O nó selecionado é exibido no painel
  • ✅ As sugestões são geradas sem duplicar tópicos existentes
  • ✅ As sugestões podem ser adicionadas ao mapa mental
  • ✅ O modo personalizado permite inserir prompts customizados
  • ✅ O indicador de carregamento aparece durante a geração

⚠️ Possíveis problemas:

  • Se receber erro de CORS, verifique se a edge function está configurada corretamente
  • Se receber erro de API key, confirme se a OPENAI_API_KEY está configurada no Supabase
